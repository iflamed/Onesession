Onession 简介

Onession 致力于解决在同一个域名的后端使用多个语言进行开发时共享session 的问题。多个语言前端可以使用nginx 进行代理。nginx 可以根据url转发相应请求到不同的语言环境下。由此后端可以取到对应的session cookie，使用这个cookie 作为key 就可以在存储中心取到 session 数据。目前支持的存储中心只有Memcache，很快基于redis 和 数据库的方案就会出来。能够集成的语言环境目前也只有php 和 nodejs。
下一步将会支持的语言是python和java。

PHP 开发环境

PHP 开发环境只需要通过composer 安装 iflamed/onesession 包即可简单使用。你可以通过在composer.json 中require iflamed/onesession 即可 或者 直接安装。

$ composer require iflamed/onesession:dev-master

下面是一个通过composer 安装iflamed/onession 的demo 。 

  
        require 'vendor/autoload.php';
        $cacheConfig = array(
            'config'=>array(
                array(
                        'host'=>'127.0.0.1',
                        'port'=>11211,
                        'weight'=>10,
                ),
            ),
            'useMemcached'=>false,
        );
        $keyPrefix='localhost';
        $storeClassName = 'MemcacheStore';
        Onesession\HttpSession::init($storeClassName,$cacheConfig,$keyPrefix);
        session_start();
        $_SESSION['serialisation'] = 'should be in json';
        if (!isset($_SESSION['a'])) {
            $_SESSION['a'] = 0;
        }
        $_SESSION['a']++;
        var_dump($_SESSION);

Nodejs 开发环境

Nodejs 开发环境需要使用onesession 包。这个包是一个完整的connect middleware。所以完全兼容express、sail.js 等使用connect 处理session的环境。

$ npm install onesession

下面是一个使用 onesession 例子


  var express = require('express');

  // pass the express to the connect memcached module
  // allowing it to inherit from express.session.Store
  var MemcachedStore = require('connect-memcached')(express);
  var onesession = require('onesession');
  var app = express();

  app.use(express.favicon());

  // request logging
  app.use(express.logger());

  // required to parse the session cookie
  app.use(express.cookieParser());

  // Populates:
  // - req.session
  // - req.sessionStore
  // - req.sessionID (or req.session.id)

  app.use(onesession({
    secret: 'not used',
    store: new MemcachedStore({
      hosts: [ '127.0.0.1:11211' ] // Change this to your memcache server(s). See Options for additional info.
    })
  }));

  app.get('/', function(req, res){
    if (req.session.views) {
      ++req.session.views;
    } else {
      req.session.views = 1;
    }
    res.send('Viewed <strong>' + req.session.views + '</strong> times.');
  });

  app.listen(3000);
  console.log('Express app started on port 3000');
